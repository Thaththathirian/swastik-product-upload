# PMS Admin | Product Management System

Professional product management application built with modern technologies.

## Features

- **Product Management**: Create, edit, and delete products.
- **Media Support**: Sequential image cropping and sorting.
- **Inventory Control**: Track stock levels and status.
- **Responsive Design**: Fully optimized for mobile and desktop.

## Technologies

- **Vite** & **React**
- **TypeScript**
- **Tailwind CSS**
- **shadcn/ui**
- **Lucide Icons**

## Getting Started

### Prerequisites

- Node.js (v18 or later)
- npm or bun

### Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/Thaththathirian/swastik-product-upload.git
   ```

2. Install dependencies:
   ```sh
   npm install
   ```

3. Start development server:
   ```sh
   npm run dev
   ```

## Deployment

Build the production bundle:
```sh
npm run build
```

The app can be deployed to Vercel, Netlify, or any static hosting service.
